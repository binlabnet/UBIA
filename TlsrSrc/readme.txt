1) Скачать с github UBIA: git clone https://github.com/pvvx/UBIA
2) Открыть Telink IDE 
3) В окне "Project Explorer" правой кнопкой мышки выбрать: "Import->General->Existing Projects into Workspace"
4) Указать скачанный каталог "TlsrSrc" в UBIA
5) Поставить галку "Copy projects into workspace" и нажать "Finish"

Далее, по удобству, можно закрыть проект "tl_pvvx_ble_sdk" чтобы не мешался.
Возможно снять галку в меню "Project->Build Automatically"...
